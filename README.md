 **# CustomSpray** 
___

## 介绍
___
允许玩家提供 图片(文件/链接) 在服务器中展示喷涂

因兼容需要，率先开发 [?~1.12] 版本 （原因：1.13及以上的版本将允许展示框在方块上下两端放置）

## 特性
___
- 无 Bungeecord 端插件，所有数据皆由 MySQL/yml 进行存储

- [待改善] 仅支持 128*128px 图片
