## ✨ 用法 | Usage
___
### 步骤
```
    1 | 点击上面的 【下载 | Dowmload】 链接下载 .zip 压缩包
      |
    2 | 解压，与你的oraxen pack目录合并
      |
    3 | 输入指令: /o reload all  -  重新打包资源包
      |
    4 | 修改配置:
      |
      | 找到 plugins/customsprays/config.yml 中的:
      | spray_sound: "default-1-0.8"
      |
      | 改为:
      | spray_sound: "customsprays:sprays-1-1"
```
